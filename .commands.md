Bot Command List*

🛠️ General Commands
```
!help              - Show all commands  
!menu              - Display interactive menu  
!ping              - Check bot speed  
!owner             - Bot owner's contact  
!report [msg]      - Report an issue or bug  
!rules             - Show group or bot rules  
```

👑 Admin Tools
```
!tagall            - Mention everyone in the group  
!groupinfo         - View group details  
!promote @user     - Make member an admin  
!demote @user      - Remove admin rights  
!kick @user        - Remove a user from group  
!add [number]      - Add member to group  
!open              - Unlock group for messages  
!close             - Lock group (admin only)  
!revoke            - Reset invite link  
```

🎭 Media Tools
```
!sticker           - Convert image/video to sticker  
!toimg             - Convert sticker to image  
!tomp4             - Convert sticker to video  
!toaudio           - Convert video to audio  
!tomp3             - Convert voice to MP3  
!removebg          - Remove background from image  
```

🎵 Music & Video
```!play [name]       - Play and send song  
!ytmp3 [url]       - Download YouTube audio  
!ytmp4 [url]       - Download YouTube video  
!lyrics [name]     - Show song lyrics  
```

🌍 Internet Tools
```
!google [query]    - Search Google  
!weather [city]    - Show current weather  
!wiki [topic]      - Get Wikipedia summary  
!covid [country]   - Covid-19 stats  
```

🧠 AI & Utility
```
!tts [text]        - Convert text to speech  
!translate [lang] [text] - Translate text  
!qr [text]         - Generate QR code  
!calc [equation]   - Solve math problem  
```

🎉 Fun & Random
```
!meme              - Send a random meme  
!joke              - Tell a joke  
!truth             - Truth game prompt  
!dare              - Dare game prompt  
!fact              - Random fun fact
```
